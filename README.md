this is the projects repository for iheartcomputer.club

see the folders: iheartccomputer or vim-demo for some examples of how this works but my idea is to just write a simple webpage for all the projects we do, this way they can be easily accessed during and after presentations

i am in the process of deploying 'actions' on ryanhub so soon enough any contributor/maintainer on this repo will be able to create their own projects and push them to the main site!
