# box-physics

An extremely simple and naive physics engine for boxes only. This project is not designed to be a production-grade physics engine.
It is intended to be a learning project for others who are interested in physics engines and want to see the beginning of  implementation.

The physics engine is implemented in C using [cglm](https://github.com/recp/cglm) for the math and [raylib](https://www.raylib.com/) for the test-client rendering.
It supports basic physics simulation, including gravity, collision detection, and collision response for axis-aligned boxes.
